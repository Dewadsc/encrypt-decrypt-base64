File "main.py" berfungsi untuk menampung kata kata yang ingin kalian encrypt.

File "tools.py" berfungsi untuk mengeksekusi encrypt dan disimpan kedalam file yang nantinya muncul secara otomatis dengan nama "hasilEncryptBase64.py".

File "hasilEncryptBase64.py" berfungsi untuk mendecrypt atau mengembalikan kata asal.
